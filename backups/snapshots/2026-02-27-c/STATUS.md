# STATUS

## Поточний стан

- Єдиний вхідний UI: `root/index.html`.
- Працюють 3 курси: `english`, `spanish`, `russian`.
- Є меню вправ із круглими кнопками, прокруткою та підсвіткою вправи з активним прогресом.
- Відновлення прогресу відкривається по кліку на кружечок вправи з прогресом.

## Як працює сценарій користувача

1. Користувач відкриває курс.
2. Рушій завантажує `manifest.json` і файли `exN.js` у `blocks`.
3. На екрані “Вправи” можна обрати вправу:
   - Для вправи без збереженого прогресу відкривається прев'ю і старт по сценарію.
   - Для вправи зі збереженим прогресом доступне продовження/перезапуск.
4. У main-режимі проходження йде по 3 етапах (вибір, вписати, речення).
5. Прогрес і стан етапів зберігаються в `localStorage`.

## Ключові файли логіки

- `shared/js/app.js` — основний state machine, resume/restart, модалки, XP/етапи.
- `root/index.html` — layout, стилі, прив'язка UI-кнопок та сторінок.
- `*/js/course_config.js` — текстові підписи та локалізація.

## Що перевіряти після змін

- Відкриття `root/index.html` через веб-сервер.
- Перехід у “Вправи”, вибір кружечків, resume/restart сценарій.
- Збереження/відновлення прогресу після перезавантаження сторінки.
- Відображення тем (`classic`, `cyberpunk`, `kawaii`).
